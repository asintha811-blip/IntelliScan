# IntelliScan

## AI-Enhanced Web Application Vulnerability Scanner

IntelliScan is a lightweight and intelligent web application vulnerability scanner developed as a final year project. It is designed to identify common security misconfigurations and potential vulnerabilities using safe, structured, and heuristic-based analysis techniques.

---

## 🔍 Key Features

- 🔐 Security Header Analysis (CSP, HSTS, X-Frame-Options, etc.)
- 🧪 Reflected Input Detection (XSS indicators)
- 💉 SQL Injection Heuristic Detection
- 🖥️ Server Information Disclosure Detection
- 📊 Intelligent Risk Scoring System
- 🧠 Confidence-Based Findings:
  - Confirmed
  - Possible
  - Observed
- 📄 Exportable Reports:
  - PDF
  - CSV
  - TXT
- 🎨 Modern Glassmorphism UI

---
